# Data

## Real Source

The starter uses World Football Elo Ratings from:

```text
https://www.eloratings.net/
```

The main downloaded table is:

```text
https://www.eloratings.net/World.tsv
```

Additional downloaded files include `en.teams.tsv` for team names and confederation pages such as `UEFA.tsv`, `CONMEBOL.tsv`, `CONCACAF.tsv`, `CAF.tsv`, `AFC.tsv`, and `OFC.tsv` for dashboard filters.

Yearly result files named `elo_results_*.tsv` are downloaded by `scripts/calibrate_draw_rate.py` from URLs such as:

```text
https://www.eloratings.net/2022_results.tsv
```

Those files include match scores and pre-match Elo ratings, which are used to fit the group-stage draw probability curve.

## Derived Starter Ratings

`derived_team_strengths.csv` and `derived_team_strengths.json` are generated by:

```bash
PYTHONPATH=src python3 scripts/fetch_elo_data.py
```

The exported `elo` and `rating` fields preserve the current World Football Elo rating. The `attack_rating` and `defense_rating` columns are starter-only compatibility fields derived from historical goals for and against per match.

These ratings are a data-backed prototype input, not a calibrated tournament model.
